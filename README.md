# 🚀 Skill Planner Pro

A modern, responsive web application that helps students plan their career path, manage daily study tasks, and track their progress.

---

## 📌 Features

### 🔐 Authentication
- User Registration & Login
- Data stored using LocalStorage
- Basic validation system

### 🏠 Dashboard
- Central navigation panel
- Access to Planner, Progress & Career Finder

### 📅 Skill Planner
- Add daily study tasks
- Mark tasks as completed ✔
- Delete tasks ❌
- Tasks stored in browser storage

### 📊 Progress Tracker
- Displays completion percentage
- Dynamic progress bar
- Auto updates based on tasks

### 🧭 Career Finder (Main Feature)
- Search career options like:
  - Engineering
  - Pharmacy
- Displays:
  - Branches
  - Required Skills
  - Fees structure 💰
  - Career roadmap 📚

### 🌙 Dark Mode
- Toggle dark/light theme
- Improves user experience

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- Bootstrap 5  
- JavaScript (Vanilla)  
- LocalStorage
